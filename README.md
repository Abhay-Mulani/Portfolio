---

## 🚀 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/abhay-mulani/Portfolio.git
   ```

2. Navigate to the project folder:
   ```bash
   cd Portfolio
   ```

3. Open `index.html` in your browser:
   ```bash
   open index.html   # macOS
   start index.html  # Windows
   ```
